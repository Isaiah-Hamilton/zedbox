## Features

- **Native Zed Support** - Built specifically for the Zed editor
- **Dark & Light Themes** - Easy on the eyes for late night or early morning coding sessions

### Manual Installation

1. Clone this repository:
   ```sh
   git clone https://github.com/isaiah-hamilton/zedbox.git
   ```
2. Copy the `themes/zedbox.json` file to your Zed themes directory

## Variants

Currently available:

- **Zedbox Material Hard Dark** - A dark theme with high contrast based on Gruvbox Material

Planned variants include:
- Material Soft/Medium (Dark & Light)
- Original Gruvbox (Soft/Medium/Hard, Dark & Light)
- Mix variants
